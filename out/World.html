draw()

This feature draws game action on the screen based on the current game status. If the game is not over, the screen will be blanked, the camera will move according to the game situation, objects will be added, and the background sound will play. If the game is over and the character is dead, objects are added for ending the game, otherwise objects are added for winning the game. The function then calls itself again with requestAnimationFrame to allow continuous updating of the screen.
